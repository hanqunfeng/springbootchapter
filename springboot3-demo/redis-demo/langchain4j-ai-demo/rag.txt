问题：Redis 7 如何开启向量检索功能？
答案：需要安装并加载 RediSearch 模块，然后创建包含 VECTOR 字段的索引即可支持向量相似度搜索。

问题：Redis 中 STRING 和 HASH 的主要区别是什么？
答案：STRING 适合存储单值或序列化对象，HASH 适合存储结构化字段，支持单字段读写和节省内存。

问题：Spring Boot 如何配置 RedisTemplate 连接 Redis 集群？
答案：需要配置 cluster.nodes 节点列表，并使用 Lettuce 客户端自动发现和维护集群拓扑。

问题：Java 中如何计算两个向量的余弦相似度？
答案：可以通过计算向量点积并除以两个向量的模长来得到余弦相似度。

问题：GraalVM Native Image 构建时常见的问题有哪些？
答案：主要包括反射配置缺失、动态代理不兼容、资源文件未打包以及第三方库不支持。

问题：Kubernetes 中 Pod 和 Deployment 的区别是什么？
答案：Pod 是最小运行单元，Deployment 用于管理 Pod 副本、滚动升级和自动恢复。

问题：Redis 的事务是否保证强一致性？
答案：Redis 事务只保证命令顺序执行，不支持回滚，也不具备传统数据库的强一致事务语义。

问题：Redis 中如何实现分布式锁？
答案：通常使用 SET NX PX 命令设置带过期时间的 key，并通过 Lua 脚本保证原子释放。

问题：Spring Boot 项目中如何优雅关闭线程池？
答案：可以在容器关闭时调用 ExecutorService 的 shutdown 和 awaitTermination 方法。

问题：什么是向量数据库的 ANN 检索？
答案：ANN 是近似最近邻搜索，通过牺牲少量精度换取大规模向量检索的高性能。

问题：MySQL 索引为什么会降低写入性能？
答案：因为每次写入都需要同步维护索引结构，会增加额外的磁盘和 CPU 开销。

问题：Redis 为什么适合做高并发缓存？
答案：因为其基于内存、单线程事件模型和高效的数据结构，延迟极低。

问题：Kubernetes Service 的作用是什么？
答案：Service 提供稳定访问入口，实现 Pod 的负载均衡和服务发现。

问题：什么是 JVM 的垃圾回收暂停？
答案：垃圾回收时应用线程被暂停执行，称为 Stop The World，会影响系统响应时间。

问题：Spring 中 Bean 的生命周期包含哪些阶段？
答案：包括实例化、属性注入、初始化回调、使用阶段以及销毁回调。

问题：Redis Cluster 如何进行数据分片？
答案：通过 16384 个哈希槽将 key 分布到不同节点，实现水平扩展。

问题：什么是 Embedding 向量？
答案：Embedding 是将文本或对象映射为高维数值向量，用于语义相似度计算。

问题：Kubernetes 中如何限制容器资源使用？
答案：可以在 Pod 中配置 CPU 和内存的 requests 和 limits 参数。

问题：数据库连接池的核心作用是什么？
答案：复用数据库连接，减少频繁创建和销毁连接的开销，提高系统吞吐能力。

问题：Redis 中 Lua 脚本的优势是什么？
答案：可以保证多命令原子执行，减少网络往返并提升一致性。
